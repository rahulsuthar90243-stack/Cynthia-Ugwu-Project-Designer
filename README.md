# Cynthia Ugwu Portfolio Clone

A modern portfolio website clone inspired by Cynthia Ugwu's official website.

##  Project Overview

This project is a front-end portfolio clone built to practice advanced UI design, smooth animations, and modern web development techniques.

The goal of this project was to improve skills in:
- HTML structure
- Advanced CSS styling
- JavaScript animations
- GSAP animation library
- Locomotive Scroll
- Custom cursor effects
- Infinite scrolling logo section

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP (GreenSock Animation Platform)
- Locomotive Scroll
- Remix Icons

---

##  Features

- Smooth scrolling effect
- Custom animated cursor
- GSAP intro animations
- Hover underline animations
- Image hover effects
- Infinite auto-scrolling logo section
- Responsive design
- Modern dark theme UI

---

##  What I Learned

- How to use GSAP timelines and stagger animations
- How to implement smooth scrolling with Locomotive Scroll
- Creating custom cursor effects
- Building infinite scrolling sections
- Writing clean and structured front-end code

---

##  Purpose

This project was built for learning and practice purposes to improve front-end development skills.
